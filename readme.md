# 💪 AI Wellness Coach

An AI-powered wellness assistant that helps users with fitness planning, diet guidance, progress tracking, and smart health insights using LLMs.

---

## 🚀 Features

- 🤖 AI Chatbot for fitness & diet queries  
- 📅 Weekly fitness & diet plan generation  
- 📊 Weight tracking dashboard with charts  
- 🧠 Smart insights (trend & plateau detection)  
- 📄 PDF report generation  
- 👤 User profile with BMI calculation  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- OpenAI API (via OpenRouter)  
- ReportLab  

---

## ▶️ How to Run

1. Clone this repository on your computer
2. Install dependencies: pip install -r requirements.txt
3. Add your API key in `.env` file:
4. Run the app: streamlit run app.py

## 📊 Features in Detail

### 🤖 AI Chatbot
- Answers user queries related to fitness, diet, and health

### 📅 Weekly Planner
- Generates personalized 7-day workout & diet plans

### 📊 Progress Dashboard
- Tracks daily weight entries
- Displays trends using charts

### 🧠 Smart Insights
- Detects weight trends
- Identifies plateau
- Provides actionable feedback

### 📄 PDF Report
- Generates a complete wellness report including:
  - User details  
  - Weight history  
  - Weekly plan  
  - AI insights
